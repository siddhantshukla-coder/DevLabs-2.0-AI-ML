# Project Function Calling

A simple AI Agent built using the Gemini API that demonstrates how Large Language Models (LLMs) can use external tools through function calling.

## Overview

This project implements a basic ReAct-style (Reason + Act) agent from scratch without using frameworks such as LangChain or CrewAI.

The agent can:

* Retrieve real-time weather information for a city.
* Retrieve country information using an external API.
* Decide when a tool is needed.
* Execute tools automatically.
* Feed tool results back to the LLM.
* Generate a final response based on tool outputs.

## How It Works

The agent follows a ReAct-style loop:

1. User sends a message.
2. Gemini analyzes the request.
3. Gemini decides whether a tool is required.
4. If a tool is needed:

   * Gemini returns a function call.
   * Python executes the corresponding tool.
   * Tool output is sent back to Gemini.
5. Gemini produces the final answer.

## Tools

### Weather Tool

Uses the wttr.in API to fetch real-time weather information.

### Country Information Tool

Uses the Rest Countries API to fetch country details.

## Sample Runs

### Example 1: Weather Lookup

**User Query**

```text
What is the weather in Tokyo right now?
```

**Tool Call Generated by Gemini**

```text
get_weather(city="Tokyo")
```

**Tool Output**

```text
Tokyo: 🌤️ +20°C
```

**Final Response**

```text
The weather in Tokyo right now is 🌤️ +20°C.
```

---

### Example 2: Country Information

**User Query**

```text
Tell me about Japan.
```

**Tool Call Generated by Gemini**

```text
get_country_info(country="Japan")
```

**Tool Output**

```json
{
  "name": "Japan",
  "capital": "Tokyo",
  "population": 125000000,
  "region": "Asia"
}
```

**Final Response**

```text
Japan is a country in Asia with Tokyo as its capital. It has a population of approximately 125 million people and is known for its rich culture, advanced technology, and beautiful landscapes.
```

---

### Example 3: Travel Recommendation

**User Query**

```text
Should I visit Tokyo next week?
```

**Reasoning Process**

```text
Gemini determines that current weather information may help answer the question.
```

**Tool Call**

```text
get_weather(city="Tokyo")
```

**Tool Output**

```text
Tokyo: 🌤️ +20°C
```

**Final Response**

```text
Based on the current pleasant weather conditions in Tokyo, visiting next week could be a great choice. However, you should also consider your budget, travel goals, and accommodation availability before making a final decision.
```
