AI Travel Assistant Agent

Features
- Search tourist attractions
- Check current weather for any city
- Suggest packing items based on weather conditions

Tools Used
1. Tourist Attraction Search: Uses DuckDuckGo Search Tool to find places to visit.
2. Weather Tool: Fetch weather info for a city using wttr.in.
3. Packing Advisor: Suggests what to pack based on weather conditions.

Tech Stack
- Python
- SmolAgents
- Hugging Face Inference API
- Requests

How It Works
1. User enters a travel-related question.
2. The agent reasons about the question.
3. Agent calls tools which are required.
4. The agent takes action and generates a final answer.
