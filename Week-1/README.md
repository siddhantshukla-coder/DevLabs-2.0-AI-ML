# Week 1 — Agentic AI Basics

Build a simple Agentic AI app using any framework (LangChain, CrewAI, AutoGen, LlamaIndex, etc.)

## Requirements
- Agent takes a goal as input and breaks it into steps autonomously
- Uses at least one tool (web search, calculator, API call, etc.)
- Include a README with what it does, how to run it, and sample output

## Submission
Go into Week-1/ and create a folder with your name. Add your agent.py, requirements.txt, and README.md inside it.
