# DevLabs 2.0 — AI/ML Track 🤖

Welcome! This is where mentees submit their weekly Agentic AI tasks.

## 🚀 How to Submit
1. Fork this repo
2. Go to the current week's folder (e.g. Week-1/)
3. Create a folder with your name inside it
4. Add your files inside your folder
5. Open a Pull Request titled: [Week-1] Your Name

## 📌 Guidelines
- One folder per mentee per week
- Include a README.md explaining what your project does
- Do not modify other mentees folders
